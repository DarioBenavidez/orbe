# Orbe - Finanzas Personales

Monorepo con el frontend mobile y el backend del bot de WhatsApp.

## Estructura

- `frontend/` — App móvil (React Native / Expo)
- `backend/` — Bot de WhatsApp (Node.js + Express + Twilio + Supabase)

## Frontend

```bash
cd frontend
npm install
npx expo start
```

## Backend

```bash
cd backend
npm install
npm start
```
